About SkillMatch AI

An intelligent resume ranking system built for HR professionals

What is SkillMatch AI?

SkillMatch AI is a web application that helps HR professionals find the right candidates faster. It extracts text from uploaded resumes, analyzes skills and education, then ranks candidates based on how well they match the job requirements — saving hours of manual screening.

How it works

1
Upload resumes
Upload PDF files for all candidates at once
2
Set requirements
Enter required skills and minimum education level
3
AI analysis
The system extracts name, email, phone, education, and skills from each resume
4
Ranked results
Candidates are scored and sorted by match percentage

Python Flask SQLite PyPDF2 pdfplumber HTML / CSS JavaScript
